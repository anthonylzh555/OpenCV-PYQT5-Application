# OpenCV-Threshold-Application
## Just making something with user interface
 Requirement 
 - pyqt5 == 5.13.0
 - opencv-python == 3.4.1
 - numpy == 1.17.2
******
 Static
 - open certain img
 - cvtCOLOR_BGR2GRAY
 - draw a rectangle and cut 
 - save picture
 - set threshold value with scroll bar
 - calculate the pixel of the threshold
 
 Dynamic
 - open the camera with buttom
 - instant threshold 
 - instant mask
 - save initial doc
 - load initial doc
